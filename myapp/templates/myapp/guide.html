User Guide

This guide will help users and maintenance personnel operate and maintain ACME's machinery safely and efficiently.

1. Logging Into the System

  • Navigate to the Login page.
  • Enter your assigned username and password.
  • Click “Log in” to access the dashboard.

2. Viewing Machine Status

  • Use the navigation bar to open the Machine Description page.
  • Each section provides images, videos, and details of current machine models.
  • Status indicators may be integrated into future dashboards.

3. Reporting Issues

  • Contact the maintenance team directly for urgent repairs.
  • Include machine name, issue description, and urgency in all communication.
  • A form-based submission system will be added soon.

4. Safety Guidelines

  • Always wear protective gear while operating machinery.
  • Do not bypass safety interlocks.
  • Report malfunctioning machines immediately.
